Instructions:

1. Click "Draw Walls" or "Draw Paths" then click on cells to edit

2. Click "Set Start" or "Set End" then click a cell to place them

3. Click "Solve Maze" to see if there's a solution (highlighted in yellow)

4. Use "Reset Bot" to return the bot to the start position

5. Use "Clear Maze" to start over

6. Select a new size and click "Apply Size" to change maze dimensions

Keyboard Shortcuts:

W - Draw Walls

P - Draw Paths

S - Set Start

E - Set End

Space - Solve Maze

R - Reset Bot

C - Clear Maze
